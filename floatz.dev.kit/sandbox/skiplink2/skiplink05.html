Skiplink demo - 05 Fixing the problem in IE 5.5, 6, 7, Firefox and Chrome

To fix the problem when using anchors in combination with javascript, we only have to add tabindex="-1" to all anchors. In that case, also the anchors can be focused correctly. This seems to be the solution that works in all browsers (as currently know). The good news is, that now it is also possible to navigate to areas in the page that have no visible hyperlink. Only use anchors and everything is ok.
