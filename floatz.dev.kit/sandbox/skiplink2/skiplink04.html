Skiplink demo - 04 Fixing the problem in IE 5.5, 6, 7, Firefox and Chrome

This solution uses javascript and works in every browser. When selecting an element via ID a javascript function is called, that sets the focus manually. The only drawback is, that this does not work with simple internal anchors (IE 5.5, 6, 7 and Chrome). The target items must be fully hyperlinks. So the remaining question is, what to do if the target area has no link and only content - In that case the solution works in NO browser!?
